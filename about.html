About Autism Help

The Autism Help is an interactive Web GIS application designed to help users find and explore autism support services across Maryland. The system integrates spatial data with an intuitive user interface to improve access to essential services such as schools, hospitals, therapy centers, and support organizations.
The application allow users to visualize and interact with service locations on the map. It includes a layer control system that enables users to filter services by category, making it easier to focus on specific types of support. Users can search for services by name, city, address, or category using a built-in search bar, and results are displayed dynamically. When a service is selected either from the map, search results, or nearby list— the service details panel displays key information such as name, address, category, and city. The app also includes a “Find Nearby Services” feature, which uses the user’s current location to identify and rank the closest services based on distance. This supports basic spatial analysis by evaluating geographic accessibility. Additionally, users can save services, which are stored locally in the browser for easy access in future sessions. To improve usability, the application supports multi-language functionality (English and Spanish), allowing users to switch languages dynamically.
It was developed as part of a Capstone Project.

Data Source

The data used in this application consists of autism-related service locations, including specialized schools, therapy centers, hospitals, and support organizations. The dataset was collected from publicly available sources and processed to ensure consistency and accuracy. Data cleaning involved removing duplicates, standardizing attributes, and verifying location information. The final dataset was converted into GeoJSON format for integration with the Leaflet web mapping application.
